Additional paragraph. Retro computing is a wonderful hobby that connects us with the history of personal computing. The Atari 8-bit line remains one of the most beloved platforms.

The Atari 800XL was released in 1983 and quickly became the most popular model in the 8-bit Atari line. With 64KB of RAM, ANTIC and GTIA custom chips, and the legendary SIO bus, it was a powerhouse for its time.

Today, modern expansions like VBXE add 80-column text, 640x480 graphics with 256 colors, and hardware overlay capabilities. Combined with FujiNet for WiFi networking, the old Atari can browse the web!

  • Feature 1: Word-wrapped text rendering at 80 columns
  • Feature 2: Clickable links with ST mouse support
  • Feature 3: Image viewing via VBXE blitter
  • Feature 4: HTML comment and entity support
  • Feature 5: Heading skip navigation with H key

This browser is written entirely in 6502 assembly language using the MADS assembler. The code is organized into modules: parser, renderer, HTTP client, FujiNet driver, VBXE driver, mouse driver, and UI.

VBXE Board Details: The Video Board XE is an FPGA-based video expansion that plugs into the ANTIC socket. It provides 512KB of dedicated video RAM, hardware blitter for fast block transfers, and multiple overlay modes.

In text mode, VBXE can display 80 columns with per-character color attributes using a 256-entry palette. Each character position has a char byte and an attribute byte, allowing rich text formatting.

FujiNet Details: FujiNet replaces the SIO cable with a WiFi-enabled smart device. It emulates disk drives, printers, modems, and provides network access through the N: device handler.

HTTP requests are made by opening N:http://url with the appropriate SIO commands. FujiNet handles DNS resolution, TCP connections, and HTTP protocol internally.
